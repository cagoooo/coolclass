# 🌈 阿凱老師的教室小幫手

> 桃園市龍潭區石門國民小學・五個給班級用的小工具

完全靜態（純 HTML / CSS / JS），打開檔案就能用，資料存在瀏覽器的 `localStorage`。
不需要登入、不會把資料上傳，可以離線使用、可部署到 GitHub Pages。

## 五個小工具

| 檔案 | 工具 | 說明 |
|---|---|---|
| `index.html` | 🏠 統一首頁 | 五個工具的入口 |
| `emotion.html` | 😊 情緒打卡牆 | 1–30 號 + 5 種情緒（開心/生氣/難過/緊張/平靜），今日心情即時上牆 |
| `todo.html` | 📝 班級 To-Do List | 紅黃綠三欄（緊急/進行中/準備區），支援拖曳、自動存檔、匯出備份 |
| `cards.html` | 🌈 彩虹能量卡 | 103 句阿德勒風格鼓勵小語（含 3 句指定固定句），抽卡 + 收藏能量牆 |
| `attendance.html` | 📋 學生出勤記錄表 | 自訂人數、選月份自動扣週末、四種標記（出席/遲到/請假/缺席）、列印 / 存 PDF |
| `marquee.html` | 📢 跑馬燈通知 | 投影機 / 白板顯示，8 種主題、可調速度字體、全螢幕模式 |

## 使用方式

### 本機開啟
直接點兩下 `index.html` 用瀏覽器打開即可，或拖到瀏覽器視窗。

### 部署到 GitHub Pages
1. 在 GitHub 建一個 repo（例如 `coolclass`）
2. 把這個資料夾的內容 push 上去
3. Repo Settings → Pages → Source 選 `main` branch / `/ (root)`
4. 等 1-2 分鐘就會出現 `https://<你的帳號>.github.io/coolclass/`

## 操作小提示

### 情緒打卡牆
- 點自己號碼 → 點心情按鈕 → 自動貼到打卡牆
- 雙擊牆上貼紙可以取消打卡
- 預設 30 號，可在頁面內「修改學生人數」改成 1～50

### To-Do List
- 點欄位裡的「＋ 新增」加項目，標題下方可以打備註
- 拖曳卡片在欄位之間搬家或同欄內排序
- 自動存檔，也可以匯出 / 匯入 JSON 備份

### 彩虹能量卡
- 點卡片或「抽卡」鈕翻面抽一張
- 上方可以篩選類別（心態安定 / 陪伴孩子 / 教師自我鼓勵 / 教育信念 / 溫柔療癒）
- 喜歡的就「收藏到能量牆」，能量牆永久保存到 localStorage

### 出勤記錄表
- 輸入班級名稱 + 學生人數 + 月份 → 按「載入」生成表格
- 週末欄位灰底不可填，自動扣除
- 點格子可循環切換狀態（出席→遲到→請假→缺席→清空）
- 右鍵或長按格子可以叫出選單直接選
- 點「列印 / 存 PDF」用瀏覽器原生 PDF 引擎（中文不會變方框）

### 跑馬燈通知
- 在右側清單管理訊息（每則可單獨開關 / 編輯 / 刪除）
- 8 種主題、可調速度與字體大小、可自訂底色與文字色
- 點跑馬燈或按「全螢幕投影」進入投影模式，ESC 離開
- 按空白鍵可暫停 / 播放

## 注意事項

- 資料用 `localStorage` 存，**同一台電腦同一個瀏覽器**才看得到記錄
- 換電腦、清瀏覽器資料、用無痕視窗 → 舊資料不會帶過去
- 出勤記錄表建議每月「匯出 JSON」+「列印 PDF」備份
- 適合用投影機 / 觸控大屏 / iPad 操作，畫面已最佳化

## 設計理念
- **柔和彩虹色系** + 圓潤造型，適合國小學生
- **每個工具都有「← 回主頁」按鈕**，不用手動 key URL
- 全部 vanilla JS，沒有任何外部依賴或建構流程
- 響應式設計，手機 / 平板 / 桌機都能用
