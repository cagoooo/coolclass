# 🌈 教室小幫手

> 一套給老師的免費班級小工具，**歡迎所有班級老師自由使用**。

**正式網址：** https://cagoooo.github.io/coolclass/

✨ **特色：免費、免註冊、不會把資料上傳、可離線使用、可裝成 App。**

打開瀏覽器就能用，老師、學生、班長都會用。適合投影機 / 觸控大屏 / 平板 / 桌機 / 手機。

## 為什麼想做這個？

老師每天會用到一堆小工具：
- 抽籤要寫白板、計時要找手機、分組要排座位卡、出勤要翻紙本…

市面上的工具不是要付費就是要登入要綁帳號，**還可能把學生資料傳到別人雲端**。

這套工具的設計理念：

- ✅ **完全免費** — 沒有訂閱、沒有廣告、沒有付費方案
- ✅ **資料留在你電腦** — 不需要伺服器，學生資料只存在你的瀏覽器
- ✅ **不用註冊** — 打開網址就能用
- ✅ **離線可用** — 安裝成 PWA 後沒網路也能跑（戶外教學、機房沒網的時候特別好用）
- ✅ **設計給老師** — 一次設定班級名單，所有工具共用
- ✅ **可印可匯出** — 出勤可以印 PDF、匯出 CSV 交教務處

## 怎麼用？

### 方式一：直接打開網頁
1. 瀏覽器打開 https://cagoooo.github.io/coolclass/
2. 點「👥 班級名單設定」輸入班級名稱和學生姓名（從 Excel 一欄複製貼上也可以）
3. 開始用！

### 方式二：裝成 App（推薦）
- **Chrome / Edge**：網址列右邊會有「安裝」圖示，點一下就裝好
- **iPhone Safari**：點分享 → 加入主畫面
- **Android Chrome**：選單 → 加到主畫面

裝完後：
- 主畫面會出現可愛的彩虹圖示
- 開啟比網頁快很多
- **沒網路也能用**

### 方式三：自己改、自己掛
這專案是 MIT 開源，clone 下來改顏色、改字、加功能都行。歡迎做出你自己的版本送給你們學校老師。

## 注意事項

- 資料用瀏覽器 `localStorage` 存 — **同一台電腦同一個瀏覽器**才看得到記錄
- 換電腦、清瀏覽器資料、用無痕視窗 → 舊資料不會自動帶過去
- 出勤記錄表建議每月「匯出 CSV」+「列印 PDF」備份兩份
- 適合用投影機 / 觸控大屏 / iPad 操作，畫面已最佳化
- 有新版會自動跳右下角通知（粉紅 banner），按「立刻更新」就好

## 技術設計（給想 fork 的同行老師看）

- 純 HTML / CSS / vanilla JS，**無建構流程、無框架、無外部依賴**（僅 Google Fonts）
- localStorage 用 `akai_*_v1` 命名 namespace（前綴可全 repo replace 改成自己的）
- Service Worker 走 **network-first**，BUILD_VERSION 占位字串確保更新一定被偵測
- 響應式設計，手機 / 平板 / 桌機 / 觸控大屏都能用
- 所有頁面都有「← 回主頁」按鈕，避免使用者迷路
- 部署：GitHub Pages（推 main branch 就會自動上線）

### 自己改完要怎麼部署？

```bash
git clone https://github.com/cagoooo/coolclass.git
cd coolclass
# ...改完你想改的...
bash bump-version.sh "改了什麼"
git add -A && git commit -m "改了什麼" && git push
```

部署後使用者 3 分鐘內會自動收到更新通知 ✨
