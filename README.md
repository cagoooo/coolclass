# 🌈 阿凱老師的教室小幫手

> 桃園市龍潭區石門國民小學・八個給班級用的小工具

完全靜態（純 HTML / CSS / JS），打開檔案就能用，資料存在瀏覽器的 `localStorage`。
不需要登入、不會把資料上傳，可以離線使用、可裝成 PWA。

**正式網址：** https://cagoooo.github.io/coolclass/

## 八個工具

| 檔案 | 工具 | 說明 |
|---|---|---|
| `index.html` | 🏠 統一首頁 | 含今日狀態儀表板 + 所有工具入口 |
| `roster.html` | 👥 班級名單設定 | 設定一次，所有工具共用 |
| `emotion.html` | 😊 情緒打卡牆 | 1–N 號 × 5 種情緒，姓名同步顯示 |
| `todo.html` | 📝 班級 To-Do List | 紅黃綠三欄拖曳、自動存檔 |
| `cards.html` | 🌈 彩虹能量卡 | 103 句阿德勒風小語 + 每日固定今日心錨 |
| `attendance.html` | 📋 學生出勤記錄表 | 自動扣週末、PDF + CSV 雙匯出 |
| `marquee.html` | 📢 跑馬燈通知 | 8 主題 + 時鐘 + 倒數計時 |
| `picker.html` | 🎯 隨機抽號 | 公平抽人、可排除已抽過 |
| `timer.html` | ⏱️ 教室計時器 | 大字幕倒數、預設按鈕、鈴聲提醒 |
| `grouper.html` | 👥 隨機分組 | 設 N 組或每組 N 人、可印名牌 |

## v2.0 新功能

### 共用班級名單
所有需要學生資料的工具（情緒打卡、出勤、抽號、分組）都從 `roster.html` 讀取，設定一次處處可用。

### 首頁儀表板
打開 `index.html` 立刻看到：
- 今日情緒打卡數 X/Y 人
- 待辦事項總計與紅黃綠分布
- 今日能量卡（依日期固定，每天同一張）
- 今日出席人數

### PWA 安裝
首次開啟會跳出安裝提示，按「安裝」可加到主畫面當 App 用，**離線也能開**。

### 出勤表 CSV 匯出
新增按鈕「📊 匯出 CSV」，產出 Excel 可直接打開的格式：
- 含 BOM 的 UTF-8
- 每位學生每天標記 + 月底統計欄
- 自動算出席率
- 班級總計區（遲到、請假、缺席總次數）

### 跑馬燈時鐘 + 倒數
- 全螢幕模式可在左上顯示時鐘（24 / 12 小時、可選帶秒）
- 右上顯示倒數計時，可設定到指定時間或快速 +5/+10/+20/+40 分
- 倒數到 1 分鐘內變紅色閃爍提醒

### 三個課堂工具
- **隨機抽號**：大字幕轉盤、抽中音效、自動排除已抽過的（公平模式）
- **教室計時器**：8 個預設時間 + 自訂、警示色、結束鈴聲、可開滴答聲
- **隨機分組**：洗牌動畫、自動選組長、可印名牌、可複製文字版

## 使用方式

### 本機開啟
直接點兩下 `index.html` 用瀏覽器打開，或拖到瀏覽器視窗。

### 加到主畫面（推薦）
1. 用 Chrome / Edge 開 https://cagoooo.github.io/coolclass/
2. 點網址列右邊的「安裝」圖示，或同意首頁跳出的安裝引導
3. iOS Safari → 分享 → 加入主畫面

### 操作快捷鍵
- 跑馬燈 / 計時器 / 抽號：**空白鍵** = 開始 / 暫停 / 抽
- 計時器：**R** = 重設、**F** = 全螢幕、**Esc** = 離開全螢幕

## 注意事項

- 資料用 `localStorage` 存 — **同一台電腦同一個瀏覽器**才看得到記錄
- 換電腦、清瀏覽器資料、用無痕視窗 → 舊資料不會帶過去
- 出勤記錄表建議每月「匯出 CSV」+「列印 PDF」備份兩份
- 適合用投影機 / 觸控大屏 / iPad 操作，畫面已最佳化
- 服務 Service Worker 會 cache 全部資源，更新後可能要按一次 Ctrl+Shift+R 強制更新

## 技術設計
- 純 HTML / CSS / vanilla JS，**無建構流程、無框架、無外部依賴**（除 Google Fonts）
- localStorage 用 `akai_*_v1` 命名 namespace
- Service Worker 走 **network-first**，更新無痛
- 響應式設計，手機 / 平板 / 桌機 / 觸控大屏都能用
- 所有頁面都有「← 回主頁」按鈕，避免使用者迷路
